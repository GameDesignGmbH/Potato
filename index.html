Potato takes a modular approach to AS3 programming, its main goal is to improve our team's workflow and to favor code reusability in AS3 projects.

License

Potato is released under the Open Source MIT license, which gives you the possibility to use it and modify it in every circumstance.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/gringo-nu/Potato

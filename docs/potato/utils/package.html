public function construct(aClass:Class, ... args):*

Language Version : ActionScript 3
Since : 05.08.2010
Runtime Versions : AIR 1.0, Flash Player 10.0.0

Utility method for constructing a class instance dynamically. This method is a work-around for the absence of Function.apply() for class constructors.

Parameters

aClass:Class — The class you want to instantiate.
 
... args — [optional] Up to ten more arguments to be applied to the class constructor. Warning: this method is not type safe at compile time!

Returns
* — A new instance of the specified class.

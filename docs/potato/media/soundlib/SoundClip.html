Packagepotato.modules.media.soundlib
Classpublic class SoundClip
InheritanceSoundClip flash.events.EventDispatcher
Implements IMedia, IAudio

SoundClip is a composition of Sound with some important features for a simple playback.

This class was made for massive use, so it provides only the primordial controls of sound, and don't care about progress events. Can be used alone for any sound ready to play, or with custom SoundGroups.

volumeScaleproperty 
volumeScale:Number

The scale of volume relativizes the volume, controlling this power.

The volume of a SoundClid instance is a result of the volumeolumeScale. It means if the volume is 1 and the volumeScale is .7, the "real volume" will be .7. It is useful to control volume of many instances without their looses the relativity among them, like a SoundGroup do.

The default value is 1.

play2()method 
public function play2(volume:Number = -1, loops:int = 0, startTime:Number = 0, pan:Number = 0):void

Starts the sound playback with some settings like volume, loops, etc.

Useful for fast sound calls with volume an loop settings.

Parameters

volume:Number (default = -1) — The volume of playback. If the value is less than 0, the current volume setted don't change.
 
loops:int (default = 0) — The number of repetitions of the sound. If the value is less than 0, the sound will loop infinitly.
 
startTime:Number (default = 0) — The point where the playback will start.
 
pan:Number (default = 0) — The panning of the playback.
